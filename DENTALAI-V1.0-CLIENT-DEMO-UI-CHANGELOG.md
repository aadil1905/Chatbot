# DentalAI Premium v1.0 — Client Demo UI Package

## What changed

- Redesigned the Premium dashboard with a presentation-ready hero, summary cards, live schedule, lead activity, AI Coach highlight, booking activity, and payment follow-up view.
- Added a consistent visual system for rounded cards, spacing, shadows, interactions, buttons, inputs, and forms.
- Improved dashboard wording and removed broken special-character text.

## Safety

- No database migration is included.
- No WhatsApp, OpenAI, Meta, or `.env` values are included or changed.
- Existing CRM, AI Coach, WhatsApp, clinical, and billing logic stays intact.

## Install

Copy the extracted package contents into the **Premium** project root, replace files when Windows asks, then run `npm run dev`.
