# DentalAI Standard v1.0 — Package 11: Dashboard Command Centre

## Dashboard improvements

- Added quick actions for appointments, patients, invoices, and follow-ups.
- Added clickable clinic metric cards with useful destinations.
- Added a live **Today’s schedule** panel with patient, time, treatment, and status.
- Added a weekly booking activity visual.
- Added **Attention needed** alerts for overdue follow-ups and unpaid invoices.
- Added outstanding-payment and next-appointment highlights.

## Security

- Uses the existing database and WhatsApp setup; no migration or environment-variable change is required.
- `.env`, `.env.local`, `.git`, `node_modules`, and `.next` are excluded from the delivery ZIP.
