DentalAI Premium v1.0 — Package 07 installation

1. Extract this ZIP.
2. Open the extracted folder.
3. Copy its app folder into your project folder:
   C:\Users\AADIL\ai-agency\chatbot
4. When Windows asks, choose Replace files in the destination.
5. In your Git Bash terminal, run:
   npm run build
6. If the build succeeds, run:
   git add .
   git commit -m "Install DentalAI Premium Package 7"
   git push

No Prisma migration is required for this package.
