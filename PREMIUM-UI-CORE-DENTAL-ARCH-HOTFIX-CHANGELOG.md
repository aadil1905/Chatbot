# DentalAI Premium v1.0 - Real Dental Arch Hotfix

## Included

- A realistic adult dental-arch visual on the Premium dashboard.
- A realistic adult dental-arch visual in each patient clinical workspace.
- Clickable tooth targets over the arch: select a tooth, set its condition, add notes, and save to the existing patient dental chart.

## Install

Copy the extracted contents into the Premium project root and replace files when Windows asks. Then run `npm run build`.

No database migration or environment variable is required.
