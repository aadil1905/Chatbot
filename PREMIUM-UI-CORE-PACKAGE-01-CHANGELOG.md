# DentalAI Premium v1.0 - UI Core Package 01

## Included

- A reference-style, light glass dashboard: five KPI cards, appointment schedule, dental-chart workspace entry, patient summary, WhatsApp panel, and recommended clinic actions.
- Every dashboard panel links to its real workspace: appointments, leads, calendar, billing, patients, clinical workspace, conversations, follow-ups, treatment plans, and AI Coach.
- A refreshed light, clinic-ready visual system for the premium workspace.
- A functional global search page for patients, appointments, leads, and invoices.
- A functional notification bell that opens direct links to pending appointment confirmations, overdue follow-ups, and out-of-stock inventory items.
- A lighter visual sidebar, plus improved shared field, card, focus, and responsive styling.

## Installation

1. Extract this ZIP.
2. Copy its contents into the **Premium** project root, replacing files if Windows asks.
3. In Git Bash run `npm run build`.
4. Commit and push only after the build succeeds.

## Notes

- No `.env`, `.env.local`, database, token, or API key is included.
- No Prisma migration is required for this visual/UI package.
- This package must be installed on the `premium-base` branch only.
